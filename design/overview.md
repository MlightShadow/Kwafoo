# Kwafoo 系统概览

## 系统简介

Kwafoo（夸父）是一个基于AI的新闻和热点聚合系统，灵感来源于中国神话中的夸父追日故事。系统通过多种方式抓取新闻内容，利用AI技术进行智能分类和摘要生成，为用户提供个性化的新闻阅读体验。

## 核心功能

- **多源新闻抓取**：支持RSS订阅源、API接口、网页抓取等多种方式
- **AI智能处理**：自动分类、摘要生成、深度分析
- **RAG对话系统**：基于新闻数据的智能问答
- **图片处理**：自动下载、压缩和存储新闻图片
- **文本压缩**：多种算法支持，智能选择最佳压缩方案
- **Web界面**：现代化的Vue 3前端界面
- **实时监控**：任务进度和系统状态实时展示

## 技术栈

### 后端
- **Python 3.10+**：主要开发语言
- **SQLite**：轻量级嵌入式数据库
- **OpenAI SDK**：AI处理接口（支持本地模型）
- **FastAPI**：Web框架
- **BeautifulSoup4**：网页解析
- **textrank4zh**：中文文本压缩

### 前端
- **Vue 3**：前端框架
- **TypeScript**：类型安全
- **Vite**：构建工具
- **Element Plus**：UI组件库
- **Pinia**：状态管理
- **Vue Router**：路由管理
- **Axios**：HTTP客户端

## 数据流向

1. **新闻抓取**：调度器触发抓取任务 → 抓取器从RSS/API/网页获取新闻 → 存储到数据库
2. **AI处理**：调度器触发AI处理任务 → 处理器从数据库获取未处理新闻 → 调用分类器和摘要生成器 → 更新数据库
3. **用户访问**：用户通过Web界面访问 → 前端调用API → API从数据库获取数据 → 返回给前端展示
4. **RAG对话**：用户提问 → 前端调用聊天API → RAG引擎进行全文搜索 → 构建上下文 → 调用AI生成回答 → 返回给前端

## 设计原则

- **模块化**：各模块职责清晰，低耦合高内聚
- **可扩展**：易于添加新的新闻源、AI功能等
- **可维护**：代码结构清晰，文档完善
- **高性能**：支持批量处理、队列管理、缓存等
- **用户友好**：现代化的Web界面，实时反馈
